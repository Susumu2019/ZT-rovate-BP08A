# AppIO - I2C Device Scanner

## 概要

I2Cバスのスキャンアプリケーションです。マイコンに接続されたすべてのI2Cデバイスを検出し、ポート情報（ピン番号）と共にリアルタイムで表示します。

## 機能

- **複数ポート対応**: 
  - Wire（PORT.A）: SDA=GPIO2, SCL=GPIO1
  - Wire1（内部I2C）: SDA=GPIO21, SCL=GPIO22
- **デバイス認識**: 既知のデバイスを名前付きで自動表示
- **ピン情報表示**: 各デバイスの接続ポートとピン番号を表示
- **スクロール機能**: 複数デバイスはスクロールで閲覧可能
- **リスキャン機能**: タップでいつでも再スキャン実行

## 表示情報

各デバイスについて以下を表示：
- **ポート名**: Wire, Wire1など
- **アドレス**: 16進表示（0xHH）
- **デバイス名**: MPU6886など既知デバイス
- **ピン番号**: SDA/SCL のGPIO番号

## 対応デバイス

以下のI2Cデバイスを自動認識します：

| アドレス | デバイス |
|---------|---------|
| 0x68 | MPU6886 IMU |
| 0x69 | MPU6886 (ALT) |
| 0x48 | ADS1115 ADC |
| 0x58 | STAMPS3 (LEFT) |
| 0x54 | STAMPS3 (RIGHT) |
| 0x3C | SSD1306 OLED |
| 0x27 | PCF8574 I/O |
| 0x20 | MCP23017 I/O |
| 0x76 | BME280 Sensor |
| 0x40 | PCA9685 Servo |

## 使い方

1. ホーム画面でIOボタンを選択
2. アプリが起動してI2Cスキャンが自動開始
3. 接続されたデバイスがポート情報と共に表示される
4. スクロールで全デバイスを確認
5. タップで再スキャン実行

## トラブルシューティング

### デバイスが検出されない
- I2C配線（SDA/SCL）を確認
- デバイスの電源が入っているか確認
- 表示されているピン番号が実装と一致するか確認

### "Unknown"と表示される
- デバイスが既知リストにない
- AppIO.cppのidentifyDevice()に追加してください
