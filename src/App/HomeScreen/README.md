## HomeScreen - ホーム画面（アプリ一覧）

最終更新日: 2026年2月1日

### 🎯 役割
スマホのホーム画面のように、**登録された全アプリのアイコンを表示**し、タップで起動します。

### 📁 ファイル
- `HomeScreen.h` - クラス定義
- `HomeScreen.cpp` - アイコン表示とタップ検出の実装

### 表示されるタイミング
- `appManager.showHomeScreen()` が呼ばれた時
- 起動時（`initializeApps()` の後）

### 💡 動作の仕組み

1. **アイコン表示**
   - 登録された各アプリの `drawIcon()` を呼んで描画
   - グリッド状に自動配置

2. **タップ検出**
   - `TouchManager` を経由してタッチ座標を取得し、どのアイコンが押されたか判定します
   - 発見したインデックスで `appManager.switchToApp(id)` を呼んで起動します

### カスタマイズ例
- アイコンのサイズや配置を変更
- ページング機能を追加（アプリが多い時）
- カテゴリ分けを実装
